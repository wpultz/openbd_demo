CFML Components

Closest thing to a class/object as you'll find in CFML

Define a Component

Using <cfcomponent< tag in tags <cfcomponent> <!--- cfc content here ---> </cfcomponent< Using "component" keyword in cfscript 
component output="false" {
	// cfc content here
}

Data Members

Created using the "this" scope in a cfc. In tags: 
<cfcomponent>
	<cfset this.dataMember = "" />
	<!--- cfc content here --->
</cfcomponent<
In cfscript: 
component output="false" {
	this.dataMember = "";
	// cfc content here
}

Functions

Define functions inside CFCs In tags: 
<cfcomponent>
	<cffunction name="myFunc1">
		<!--- does stuff --->
	</cffunction>
</cfcomponent<
In cfscript: 
component output="false" {
	function() {
		// does stuff
	}
}
