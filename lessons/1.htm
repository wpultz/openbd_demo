How to set variables

tags

<cfset myVar = "stuff" />

cfscript

myVar = "stuff";

When to var variables

var variables when they are to be scoped inside of a function. using the "var" keyword outside of a function will cause an exception

tags

<cfset var myVar = "stuff" />

cfscript

var myVar = "stuff";

Executing functions

invoke the function as usual, or inside of a <cfset /> tag

tags

		<cfset myFunc() />

		<cfset myVar = myFunc() />

cfscript

		myFunc();
		myVar = myFunc();
